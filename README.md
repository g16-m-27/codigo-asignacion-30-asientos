# Código de Asignación de los 30 Asientos

## Descripción
Este proyecto implementa un sistema de asignación de 30 asientos utilizando estructuras de datos, Visual studio Code y C#

## Funcionalidades
- Asignación de 30 asientos en orden de llegada, una vez que todos los asientos son vendidos. Simula una línea de espera para
- una atracción en un parque de diversiones; el objetivo es asegurar que cada persona en la cola suba a la atracción en el orden correcto.
